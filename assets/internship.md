## Internship/Apprenticeship Opportunity at 1ms.ai

### About the Internship/Apprenticeship

Join our team at 1ms.ai for a rewarding experience with tons of learning on LLM, AI, Python and Rust! We are looking for enthusiastic and talented individuals to contribute to [dora-rs](https://github.com/dora-rs/dora)! This opportunity will be centered around a project to implement from beginning to the end!

#### Possible projects

- Build LLM agents that can control a Minecraft Bot!
- Build LLM agents that can control a DJI Robomaster!
- Build an auto-debugger that can fix simple python errors.
- Build a code modifier that can generate code to modify a codebase.

#### What You Will Learn

- LLM agents implementation.
- Python and Rust software engineering mentored by Rockstar Open Source Developers.
- Open Source software development.

#### Requirements

- Studying Computer Science, Math or related field.
- Experiences in Software Development: git, python, linux...
- Interest for LLM and AI.

#### Duration and Location

- Duration: 6 months for Intership and 1-2 Years for Apprenticeship
- Location: Paris (Spaces Coworking) / Remote

#### How to Apply

Please send your resume and a brief cover letter to [tao.xavier@outlook.com](mailto:tao.xavier@outlook.com) with the subject line "Internship/Apprenticeship Application".
